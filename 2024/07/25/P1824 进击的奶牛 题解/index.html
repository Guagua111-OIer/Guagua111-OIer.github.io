P1824 进击的奶牛题解

题目传送门

思路

首先，第一眼看到这个题目，你最先冒出来的想法是什么？暴力？贪心？DFS？还是……

打开一道题二话不说先看标签

这道题其实是二分。二分什么呢？其实，题目中已经有提示了：

那么，这个最大的最近距离是多少呢？

也就是说，题目要让这个距离的最小值最大。形如让最小值最大，或者最大值最小，且数据还有序的题目一般都使用二分来求解。

所以，本题解的思路被分为两部分：二分方法以及 check 函数。

二分方法

为了防止大家读代码时看着我没提到过的变量名有点懵，因为我读别人的代码时也是这样的，我做了一个列表：

然后，开始二分。

定义左边界 lll 和右边界 rrr，分别初始化为 000 和 xn−x1x_n - x_1xn​−x1​。因为 lll 最小为 000（这就不用说了吧？），rrr 最大为 xn−x1x_n-x_1xn​−x1​。因为牛棚里毕竟可能只有两头奶牛，此时距离就是 xn−x1x_n - x_1xn​−x1​。当然也可以无脑把 rrr 初始化成 2147483647。

然后 while 循环 l <= r。每次循环初始化 midmidmid 为 (l + r) / 2，然后做如下判断：

  • 如果 check(mid) 为 111，则记录答案，并令 lll 为 mid−1mid - 1mid−1（这是距离已经合法，所以 ans=midans = midans=mid，另外因为要让答案尽可能大，所以 l=mid+1l = mid + 1l=mid+1）。
  • 如果 check(mid) 为 000，则令 r=mid−1r = mid - 1r=mid−1。（这是因为 check(mid) 为 0 说明这个距离太远了，要让距离近才能够放下 ccc 头奶牛，所以调低距离）。

check 函数

接下来，我们来编写 check 函数。根据前面写的代码，check 函数的参数应该是一个距离值。于是我们先写 check 框架：

此处的 numnumnum 指的是传入的距离值，而我们 check 函数内部需要判断这个距离 numnumnum 是否合法（也就是能不能是奶牛们都相隔这个距离）。

不如把这个问题转化一下，变成：如果牛棚里的奶牛每个都相隔不少于 numnumnum 米，能不能放下所有 CCC 头奶牛。因为如果能放下所有头奶牛，使奶牛能相隔 numnumnum 米，说明 numnumnum 是合法的。（因为就这样放就可以）。但是如果不能放下 CCC 头奶牛，就说明 numnumnum 不合法。因为如果要再把剩下的奶牛挤进去，相隔距离一定会缩短，一定会有小于 numnumnum 的。

我猜肯定有人问：那如果放入放不下的奶牛后，尽管缩短距离了，但还是大于 numnumnum 呢？这是因为，前面我们已经假设过每头奶牛相互之间距离刚好是那个最小的不少于 numnumnum 的，所以此时要是放不下，那相隔距离一定会从原来的 numnumnum 缩小，导致奶牛打架。如果原来就有一些空位的话，那就不是「放不下」了。

所以，只需要再牛棚里放奶牛，每当隔间与上一个奶牛所在的隔间相差不少于 numnumnum 米，就放一个奶牛，并且计数器加一。最后只要判断计数器是不是大于等于 CCC 就可以了。
